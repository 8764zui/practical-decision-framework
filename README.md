# practical-decision-framework

> 实用主义决策框架 — 数据驱动的教育规划与职业决策助手

[![License](https://img.shields.io/badge/license-MIT-blue)](LICENSE)

## 概述

一个专注教育规划与职业决策场景的 **AI Agent 技能包**。基于结构化知识注入和 research-first 工作流，为大语言模型提供：

- **5 个核心心智模型** — 社会筛子论、选择 > 努力、就业倒推法、阶层现实主义、传播有效性
- **8 条决策启发式** — 从「连续追问」到「10年后压迫测试」的完整决策链路
- **研究优先的回答协议** — 先检索再推理，确保每一条建议都有数据支撑
- **直接高效的沟通风格** — 短句、高密度、明确判断，不模棱两可

## 使用方式

安装后，在对话中提及以下场景即可触发：

| 场景 | 示例提问 |
|------|----------|
| 选专业 | 「帮我分析一下物联网工程这个专业怎么样」 |
| 择校 | 「广东专升本，公办院校哪些能选」 |
| 职业规划 | 「做嵌入式开发和互联网后端，哪个更有前景」 |
| 决策分析 | 「我想考研但家里经济一般，怎么评估」 |
| 数据验证 | 「电气工程就业率怎么样，薪资中位数是多少」 |

也可在提问时明确指定：「用实用主义分析一下...」「数据驱动看看...」

## 心智模型

| 模型 | 一句话 | 适用场景 |
|------|--------|----------|
| 社会筛子论 | 社会用多层筛子筛选人，普通家庭能控制的只有学历 | 教育投资回报分析 |
| 选择 > 努力 | 方向错误的努力是浪费，选对赛道比拼命奔跑重要 | 重大决策方向确认 |
| 就业倒推法 | 从就业数据倒推选择，看中间50%的普通人过得怎样 | 专业 / 职业评估 |
| 阶层现实主义 | 先问家庭条件再给建议，不同背景策略完全不同 | 个性化路径规划 |
| 传播有效性 | 温吞的建议没人记住，明确判断比平衡观点更有价值 | 信息输出 |

每个模型在 `references/frameworks/mental-models.md` 中有完整的论证、案例和已知局限说明。

## 决策启发式

1. **连续追问法** — 通过结构化追问建立决策框架
2. **中位数原则** — 看中间50%，不被极端值带偏
3. **不可替代性检验** — 你的价值和你的不可替代性成正比
4. **看行动不听说辞** — 用脚投票的数据比宣传更真实
5. **家庭背景分流** — 第一条分流线：家庭经济条件
6. **城市优先原则** — 不同城市 = 不同机会密度
7. **10年后压迫测试** — 用长周期尺度倒逼决策清晰
8. **观点坚守原则** — 核心判断不退让，措辞可以调整

## 工作流

该技能采用 **Research-First** 协议：

```
收到问题
  → 分类（事实问题 / 框架问题 / 混合问题）
  → 如需数据，并行检索（就业数据 + 行业报告 + 政策信息 + 真实案例）
  → 交叉验证（至少 2 个独立来源确认）
  → 运用心智模型分析
  → 输出明确判断 + 可执行方案
```
