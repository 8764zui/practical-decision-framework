## 项目概述

本项目是一个深度定制的 **AI Agent 技能系统**，运行于 CherryStudio + OpenClaw 框架之上，专注解决**职业教育与升学规划中的信息不对称问题**。

与通用大模型的「空泛建议」不同，该系统通过长链推理、多 Agent 协作和结构化知识注入，提供**犀利、务实、高度定制化**的路径规划能力。

### 解决的核心痛点

- 职业院校学生在面临「专升本」、职业资格认证（电工证等）、间隔年规划时，政策信息散乱、路径选择迷茫
- 通用大模型回答过于笼统，缺乏针对真实教育环境的战略视角
- 关键决策点（专业选择、院校报考、时间节点规划）容易因信息滞后而错失机会
- 不同家庭经济条件下的最优策略差异巨大，需要一个能识别「阶层分流」的决策系统

### 多 Agent 协作流程

| Agent | 职责 | 技术要点 |
|-------|------|----------|
| **前置处理 Agent** | 收集并解析用户初始信息（学历、专业、地域、目标、预算） | 结构化信息抽取 + 约束条件提取 |
| **专家咨询 Agent** | 挂载 `practical-decision-framework` 技能系统，进行多步推理与沙盘推演 | 长上下文窗口 + 心智模型链式推理 + 实时联网数据验证 |
| **执行规划 Agent** | 基于推演结论生成可执行方案 | 时间线编排 + 节点预警 + 备选策略生成 |

### 核心心智模型（注入知识）

系统内置 5 个经过深度调研提炼的心智模型，作为 Agent 推理的「思维骨架」：

1. **社会筛子论** — 社会用学历、资源、平台多层筛选。普通家庭的可控变量只有学历
2. **选择 > 努力** — 方向错误的努力是浪费。选对赛道比拼命奔跑重要
3. **就业倒推法** — 从就业数据倒推选择。不看前3%的天才，看中间50%的普通人
4. **阶层现实主义** — 先问家庭条件再给建议。有试错成本和没有的，策略完全不同
5. **传播有效性** — 温吞的建议没人记住。明确的判断比平衡观点更有实际价值

| 组件 | 用途 |
|------|------|
| **CherryStudio** | Agent 运行框架、技能管理系统 |
| **OpenClaw** | 多 Agent 编排与调度 |
